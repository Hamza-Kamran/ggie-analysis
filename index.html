Overview

The data that is compiled and analyzed is retrieved from the GGIE website. User zip codes are used to geographically locate user positions and to chart and the presence and density of users. This is divided into sections: Country-wise distribution, State-wise distribution, and County-Wise distribution. Since County-wise distribution is deemed more important, we further analyze its user-density.

Interactive Map

The data presented above is translated into an interactive map that allows users to investigate the data in detail. Users can navigate across the map. Empty counties can be explored by clicking on the red markers. You can also toggle which layer you would like with the layer select button on the top right.
